# Aluminium 6063 Hybrid Metal Matrix Composite Study

## Project Overview
This academic research project investigates Aluminium 6063 hybrid metal-matrix composites reinforced with MoS2, B4C, and SiC using the stir-casting method.

## Objective
To evaluate mechanical performance improvements including:
- Wear resistance
- Creep behavior
- Corrosion resistance

## Work Performed
- Conducted wear testing under varying load conditions
- Performed creep testing for long-term deformation study
- Conducted electrochemical corrosion testing
- Analyzed microstructural behavior to correlate reinforcement composition with mechanical properties

## Key Outcome
Identified reinforcement combinations that significantly improved wear resistance and structural stability for industrial applications.

- Strength of Materials
